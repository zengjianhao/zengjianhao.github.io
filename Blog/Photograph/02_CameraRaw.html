    # Camera Raw & Lightroom 到底在调什么？

    # 基本

    # 色温

    色温指的是光线的颜色，其单位为开尔文（K）即与此光源发出相似的光的黑体辐射体所具有的开尔文温度。

    在实际应用中，只有当光源发出的光和黑体辐射的光类似时，色温才有定义。也就是，红-橙-黄-白-浅蓝这些颜色。当讨论色温时，讨论例如绿色、紫色的色温是没有意义的。

    在实际拍摄过程中，设置相机白平衡的目的是为了让相机能够准确地还原环境中的光线。当相机白平衡等于光线色温时，光线在相机中将被呈现为纯白的白光。

    在 Camera Raw 中对色温进行调整本质上是对相机白平衡进行调整，减少色温（左移）可在图像中添加冷色调（蓝色）；增加色温（右移）可在图像中添加暖色调（黄色）。

    # 色调

    设置色调以补偿绿色或洋红色色调。减少色调（左移）可在图像中添加绿色；增加色调（右移）可在图像中添加洋红色。

    # 曝光

    直方图是对图像亮度值的像素数量的统计报告图。横坐标表示0到255的亮度等级，纵坐标表示在该亮度等级下的像素数量。

    直方图由三种颜色的色条组成，它们分别表示红、绿和蓝色通道。当所有三个通道重叠时，将显示白色。当两个 RGB 通道重叠时，将显示黄色、洋红色或青色（黄色等于红色 + 绿色通道；洋红色等于红色 + 蓝色通道；青色等于绿色 + 蓝色通道）。

    如果像素的颜色值高于图像中可以表示的最高值或低于图像中可以表示的最低值，将发生修剪。系统将修剪过亮的值以输出白色（高光修剪），修剪过暗的值以输出黑色（阴影修剪）。过亮的像素被修剪后以红色叠加蒙版加以提醒，过暗的像素被修剪后以蓝色叠加蒙版加以提醒。

    调整曝光值能够调整所有像素的亮度，其单位和相机中曝光补偿一样，均为 EV。调整曝光值时，每增加 1EV，相当于将图像中所有像素的亮度增加一倍；每减少 1EV，相当于将图像中所有像素的亮度减半。

    在直方图中，降低曝光值（左移）会将直方图整体左移，调高曝光值（右移）会将直方图整体右移。

    曝光值过高会触发高光修剪，但是曝光值过低并不会触发阴影修剪。

    # 对比度

    提高对比度（左移）：直方图的两端（即最暗和最亮的部分）会变得更加突出，同时中间亮度的部分相对减少。

    降低对比度（右移）：直方图的中间部分会变得更加集中，而两端的亮度级别的分布会减少。

    # 高光

    高光只对直方图右侧（亮部）产生影响，降低高光（左移）时为了避免中间调的堆积，软件会自动提高边缘的清晰度。同时，提高高光（右移）保证不会触发额外的高光修剪。

    Tips

    • 降低高光（左移）会让亮部更多的进入中间调，从而恢复高光细节。
    • 提高高光（右移）会让亮部统一到相近的亮度水平，而且保证不触发额外的高光修剪。

    # 阴影

    阴影只对直方图左侧（暗部）产生影响，提高阴影（右移）时为了避免中间调的堆积，软件会自动提高边缘的清晰度。同时，降低阴影（左移）保证不会触发额外的阴影修剪。

    Tips

    • 降低阴影（左移）会让暗部统一到相近的亮度水平，而且保证不触发额外的阴影修剪。
    • 提高阴影（右移）会让暗部更多的进入中间调，从而恢复暗部细节。

    # 白色

    白色会对所有像素的亮度产生影响，但是越亮的区域受到的影响越大，相比于高光，降低白色（左移）并不会自动提高清晰度，并且提高白色（右移）可能会触发额外的高光修剪。

    Tips

    • 降低白色（左移）一般用于降低高光（左移）不够时的补充。
    • 提高白色（右移）一般用于恢复照片的亮部反差。

    # 黑色

    黑色会对所有像素的亮度产生影响，但是越暗的区域受到的影响越大，相比于阴影，提高黑色（右移）并不会自动提高清晰度，并且降低黑色（左移）可能会触发额外的阴影修剪。

    Tips

    • 降低黑色（左移）一般用于恢复照片的暗部反差。
    • 提高黑色（右移）一般用于提高阴影（右移）不够时的补充。
